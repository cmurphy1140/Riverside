# Riverside Cafe Website

## Project Overview
Modern, responsive website redesign for Riverside Cafe - a waterfront restaurant on the Indian River Lagoon in Vero Beach, Florida.

## Tech Stack
- HTML5 (semantic markup)
- CSS3 (custom properties, Flexbox, Grid)
- Vanilla JavaScript (no framework)
- Google Fonts (Playfair Display, Open Sans)
- Font Awesome icons

## Project Structure
```
riverside/
├── index.html          # Homepage
├── menu.html           # Menu page
├── about.html          # About/Restaurant page
├── events.html         # Events/Calendar page
├── gallery.html        # Photo gallery
├── contact.html        # Contact page
├── css/
│   ├── variables.css   # Design tokens (colors, spacing, typography)
│   ├── reset.css       # CSS reset/normalize
│   └── main.css        # Main styles
├── js/
│   └── main.js         # JavaScript (mobile nav, scroll effects)
└── images/             # Image assets
```
